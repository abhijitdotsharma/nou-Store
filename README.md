# Nou-Store

An online markeplace for all the football fans.

## What is Nou-Store ?
- It is a one stop solution for all football fantics. 
- We offer premium high quality merch sourced directly from the official Football Clubs
- Nou is a reference to Football Club Barcelona's home ground, known as Camp-nou.


## Tech Stack

- HTML
- Vanilla CSS
